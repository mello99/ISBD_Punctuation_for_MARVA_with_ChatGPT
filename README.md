# ISBD_Punctuation_for_MARVA_with_ChatGPT
Apply conservative ISBD-style punctuation normalization to selected MARC fields: 1XX/245/250/264/300/490/6XX/7XX. - ONLY add terminal ISBD punctuation (final period) IF there is NO URL/URI  anywhere in the field. If any URL/URI is present, the field is left unchanged.
